# 安全策略

## 支持范围

GoBA Slim 尚未发布稳定版本。当前仅维护 `master` 分支的最新提交；历史提交和个人分支不提供安全更新。

## 报告漏洞

请通过 GitHub 的 [Private Vulnerability Reporting](https://github.com/downdawn/goba-slim/security/advisories/new) 私下报告安全问题。

报告应尽量包含：

- 受影响的提交、版本或配置。
- 可复现的最小步骤和预期影响。
- 已知缓解方式或修复建议。

请勿在公开 Issue、Discussion、日志或示例中披露漏洞细节、利用代码、Token、Cookie、密码、私钥或其他 Secret。维护者确认问题并制定修复方案后，会协调披露时间。
